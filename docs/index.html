스마트폰이 멀어지면
자동으로 화면 보호

블루투스로 스마트폰 거리를 감지하여, 자리를 비우면 자동 화면 잠금, 돌아오면 자동 해제. 개인정보와 업무 화면을 안전하게 보호합니다.

다운로드 (무료) 기업용 둘러보기

주요 기능

설치 후 블루투스 페어링만 하면 바로 사용할 수 있습니다

📡

블루투스 근접 감지

페어링된 스마트폰의 블루투스 신호를 실시간 측정하여 거리를 판단합니다. 듀얼 소켓 방식으로 정확도를 높였습니다.

🔒

자동 화면 잠금

스마트폰이 멀어지면 자동으로 화면을 잠급니다. 잠금 지연시간, 해제 방식 등을 세밀하게 설정할 수 있습니다.

🔓

스마트 해제

스마트폰을 들고 자리에 돌아오면 자동으로 화면이 해제됩니다. 마우스/키보드로도 해제 가능합니다.

🖼️

커스텀 잠금 화면

잠금 화면에 원하는 이미지나 동영상을 표시할 수 있습니다. 기업용에서는 홍보 콘텐츠를 전사에 송출합니다.

📊

30분 타임라인

근접/이탈 이벤트를 30분 차트로 시각화합니다. 자리 비움 패턴을 한눈에 파악할 수 있습니다.

📱

자동 재연결

스마트폰이 범위를 벗어났다가 돌아오면 자동으로 블루투스가 재연결됩니다. 별도 조작이 필요 없습니다.

사용 방법

3단계로 간단하게 시작하세요

1

설치 & 페어링

SmartScreen을 설치하고 Windows 블루투스 설정에서 스마트폰을 페어링합니다.

2

기기 선택

SmartScreen에서 페어링된 기기를 선택하고 Start를 누릅니다.

3

자동 보호

이제 자리를 비우면 자동 잠금, 돌아오면 자동 해제됩니다.

기업용

전사 화면 보호 + 사내 홍보 콘텐츠 송출

중앙 관리 대시보드

  • ✅ 웹 대시보드에서 이미지/동영상 업로드
  • ✅ 업로드한 콘텐츠가 직원 PC 잠금 화면에 자동 표시
  • ✅ P2P 배포: 서버 부하 없이 사내망에서 자동 전파
  • ✅ 조직별 콘텐츠 관리 (조직 ID로 구분)
  • ✅ 보안: 직원이 자리 비우면 즉시 화면 보호
┌──────────────┐ │ 관리자 웹 │ 이미지/동영상 업로드 │ 대시보드 │ └──────┬───────┘ │ Supabase Cloud ┌──────┴───────┐ │ 첫 번째 │ 서버에서 다운로드 │ 직원 PC │ └──────┬───────┘ P2P │ 사내망 ┌──────┴───────┐ │ 직원 PC #2 │ P2P로 전파 │ 직원 PC #3 │ (서버 접속 불필요) │ 직원 PC #N │ └──────────────┘
